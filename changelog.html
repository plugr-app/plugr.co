1.0.0 — Initial public release

June 29, 2026

The first public version of Plugr. Here's everything in the box:

Library

  • Scans VST2, VST3, AU, AAX and CLAP plugins from the standard Mac folders
  • Scans your /Applications folder so you can keep all your standalone apps organized and up-to-date too.
  • Grid and sortable list views, with resizable columns
  • Filter by category, format, developer, update status, tags, or favorites
  • Tags, favorites, hide, and personal notes per plugin
  • Custom categories you can add yourself
  • Bulk edit hundreds of plugins at once (rename developer, retag, recategorize, set update source)
  • Reorderable sidebar sections via drag-and-drop
  • Fast search across all your plugins
  • Duplicate detection — spot older versions sitting next to new ones
  • One-click Move to Trash with confirmation
  • Multi-format awareness — set a tag, category, or source on one format and the others inherit it

Projects

  • Scans .als, .logicx, and .flp project files from your Ableton Live, Logic Pro, and FL Studio sessions
  • Auto-detects tempo and project key from every session
  • Three analytics charts at the top of the tab — shows you your most-used plugins, plugin categories, and developers you rely on — each clickable to filter the project list
  • Per-project ratings (A–F), customizable workflow statuses ("Rough Concept", "In Progress", "Needs Mixing", "Released"), tags, and notes
  • Filter by tempo range, key, rating, status, tags, or free-text search
  • Preview bounces (wav, mp3, etc.) auto-discovered from your project folder
  • Inline waveform player with click-to-seek, drag-to-scrub, space + arrow shortcuts, and auto-advance to the next bounce
  • Drag any audio file onto a project to attach it as a bounce; drag a single project file onto Plugr to scan just that one

Companion Apps

  • Dedicated tab showing every plugin manager on your Mac as a card grid
  • Each card displays the app icon, shows how many plugins it manages, and how many have pending updates
  • One-click launchers for Native Access, Waves Central, Splice, Plugin Alliance Installer, iZotope Product Portal, and more
  • Sortable by name, plugins managed, or updates pending

Updates

  • Built-in coverage for hundreds of developers (FabFilter, Soundtoys, Native Instruments, iZotope, Valhalla, UAD, Plugin Alliance, Cableguys, Xfer Records, and many more)
  • In-app "Find update source" flow for developers Plugr doesn't recognize — choose auto-search or manual entry, confirm in one click, and Plugr remembers for next time
  • "Major upgrade" badge for big version jumps
  • "Open companion app" status for plugins managed through vendor plugin managers
  • Edit or remove saved sources anytime
  • Opt-in community feed: working sources users submit flow back into a shared pool, making Plugr better with every update

Deals

  • Pulls live deals from trusted retailers like Plugin Boutique and Audio Plugin Deals
  • Filter to deals on plugins and developers you already own
  • Wishlist, dismiss, and sort by discount/price/newest
  • Local-currency display in USD, EUR, GBP, JPY
  • Low-price badges based on collected price history
  • Purchases made via the Deals tab may provide affiliate commissions to the developer of Plugr

Deal Alerts

  • Watch any plugin, developer, or custom keyword for matching deals — works even for plugins you don't own yet
  • Native macOS notifications when a watch fires, with 24-hour dedupe so you're never bombarded
  • Bell icons on every plugin detail, every developer name, and every deal card — flip watches on or off from anywhere
  • "N new" badge on the Deals tab counts deals you haven't seen yet
  • "My Deal Alerts…" management modal with pause / delete controls per watch
  • Optional menu-bar mode keeps Plugr alive in the background so alerts keep firing when the window's closed
  • Optional launch-at-login pairs with menu-bar mode for fully hands-off deal monitoring

Producer Tools

  • Tap tempo (with spacebar support)
  • BPM ↔ delay calculator for any BPM and any note value
  • Camelot wheel for harmonic mixing

Themes, sync, and the rest

  • Eleven themes — Light, Dark, Auto, plus eight studio palettes inspired by your favorite DAWs
  • Optional iCloud sync for tags, favorites, projects, sources, and settings
  • Backup and restore as a single file
  • CSV library export
  • Universal binary for Apple Silicon and Intel
  • Requires macOS 12 Monterey or later

Future releases will be listed above this. If you want to be notified about new versions, the app checks for updates automatically once it's installed. You can also follow along on the about page or email hello@plugr.co anytime.
